## 🛠️ Tech Stack & Skills

আমি একজন **Full Stack Web Developer** হিসেবে নিচের টেকনোলজিগুলো নিয়ে কাজ করি:

* **Backend:** PHP, Laravel (Expert), Node.js, Express.js
* **Frontend:** React.js, Next.js, Vue.js, Tailwind CSS, Bootstrap
* **Database:** MySQL, MongoDB
* **Tools:** Git & GitHub, Photoshop, Figma (Daily UI)
* **Integrations:** Stripe, Facebook API, Google API, Bulk SMS

## ✨ Key Features of this Portfolio

-   **Bento Grid Layout:** অ্যাপল স্টাইল মডার্ন কার্ড-বেজড ডিজাইন।
-   **Fully Responsive:** মোবাইল, ট্যাবলেট এবং ডেক্সটপ সব ডিভাইসের জন্য অপ্টিমাইজড।
-   **Glassmorphism:** ফ্রস্টেড গ্লাস ইফেক্ট এবং প্রিমিয়াম লুক।
-   **Optimized Performance:** হাই-স্পিড লোডিং এবং ক্লিন কোডবেজ।
-   **Project Showcase:** ক্যাটাগরি ভিত্তিক প্রজেক্ট ডিসপ্লে (Ecommerce, Hospital, Govt. ইত্যাদি)।
